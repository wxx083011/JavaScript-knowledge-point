# WebGL
WebGL 是 HTML 5 草案的一部分，可以驱动 Canvas 渲染三维场景。

WebGL 使得在支持HTML 的 canvas 标签的浏览器中，不需要安装任何插件，便可以使用基于 OpenGL ES 2.0 的 API 在 canvas 中进行3D渲染。

[three.js](https://github.com/mrdoob/three.js/) 是基于 WebGL 的一个插件。
